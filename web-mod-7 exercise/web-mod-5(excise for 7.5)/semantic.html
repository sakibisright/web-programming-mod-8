Semantic tag in Html 5

Tag Name Description
section "A section is a thematic grouping of content, typically with a heading.
article An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site
header element typically contains:one or more heading elements (h1 - h6),logo or icon,authorship information
footer A footer element typically contains:authorship information,copyright information,contact information,sitemap,back to top links, related documents
nav The nav tag element defines a set of navigation links.
aside The aside tag element defines some content aside from the content it is placed in (like a sidebar).
